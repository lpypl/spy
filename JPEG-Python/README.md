## 使用方法
### 安装
> make && make install  
> 安装目录为 `./bin`
### 编码
> lpy-jpeg-encoder.py infile outfile [info] [skip] [min_len]
### 解密
> lpy-jpeg-decoder infile [skip_count] [least_len]
